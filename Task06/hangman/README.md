# Виселица — веб-версия

Игра «Виселица» реализована как статический веб-сайт на HTML, CSS и JavaScript (ES6).  
Данные о партиях сохраняются в локальной базе данных IndexedDB.

## Возможности
- Новая игра с вводом имени игрока
- Просмотр истории всех сыгранных партий
- Повтор любой сохранённой партии с пошаговым воспроизведением

## Как играть
1. Откройте сайт: [https://olya2004.github.io/](https://olya2004.github.io/)
2. Нажмите «Новая игра»
3. Вводите по одной букве или угадывайте слово целиком

## Технологии
- HTML5, CSS3
- JavaScript (ES6 modules)
- IndexedDB (NoSQL в браузере)
